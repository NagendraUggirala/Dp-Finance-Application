# New React App

Same setup as dvskillhub: **React 19** + **Tailwind CSS** with the same theme (brand colors, container, buttons, utilities).

## Setup

```bash
cd new-react-app
npm install
npm start
```

## What’s included

- **Tailwind** – `tailwind.config.js` with brand colors, container, shadows, border radius
- **PostCSS** – `postcss.config.js` with Tailwind + Autoprefixer
- **Global styles** – `src/index.css`: `@tailwind` layers + `.btn-primary`, `.btn-outline`, `.btn-danger`, scrollbar-hide, blink/shake animations
- **Minimal app** – `src/App.js` is a placeholder; add your routes and content here

Add your content and dependencies (e.g. react-router-dom, redux) as needed.
